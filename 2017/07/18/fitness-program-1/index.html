健身计划（一） - 熟练动作热热身
发表于 2017-07-18| 更新于 2020-05-19|健身
字数总计:452|阅读时长: 1 分钟
| 阅读量:

   自从大四实习以来，由于运动明显减少，再加上管不住嘴，体重蹭蹭的涨。 BMI指数已经达到24.2，属于过重体制。摸着自己肚子上的肉，想哭的心都有┭┮﹏┭┮，所以决定趁着现在没上班，去健身房练练。但是又因为没钱请不起私教，也没有大神带我一起飞，所以只好上网搜集资料，自己制定健身计划，这才有了这篇博客。

目的

  养成运动习惯; 建立基础力量; 学习技术动作

周期

  每三天循环：有氧训练 -> 力量训练 -> 休息

内容

  关节活动 -> 10分钟慢跑热身 -> 8个动态牵拉 -> 7个肌肉激活练习 -> 当天训练内容 -> 静态牵拉

有氧训练

  跑步机/椭圆机/自行车：60分钟

力量训练

动作 强度 目的
徒手深蹲 4组，每组10次 学习技术动作
腿举 3组，每组15RM 发展下肢基础力量兼顾肌耐力
俯卧撑 4组，每组5~10次 学习技术动作
坐姿推胸 3组，每组15RM 发展上肢基础力量兼顾肌耐力
坐姿下拉 3组，每组15RM 发展上肢基础力量兼顾肌耐力
YTWL 2循环，每个动作10次 预先强化肩胛带肌群力量，稳定肩关节
交替仰卧举腿 2组，每组30次 被动发展腹部核心肌群
俯卧挺身 2组，每组45秒 发展背部核心肌群
