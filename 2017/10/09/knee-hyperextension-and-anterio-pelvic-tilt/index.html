体态矫正 - 驼背、园肩、膝关节超伸与骨盆前倾
发表于2017-10-09|更新于2020-09-01|健身
|字数总计:1.4k|阅读时长:4分钟|阅读量:

   驼背、园肩、膝关节超伸与骨盆前倾是几种常见的不良体态，容易造成颈源性头痛、膝盖疼痛，腰痛等症状。为了保持姿态平衡，还会继发性的出现 X 型腿或者 O 型腿。本文介绍了这几个矫正体态的方法，希望可以帮助大家。

驼背、园肩

   专业术语叫做“上交叉综合征”，常见于长期面对电脑的工作人员，由于缺乏运动，低头伏案，颈椎长时间不正确的姿势导致肌肉失衡，头前伸，含胸驼背，严重时会导致脊椎变形，危害健康。

膝关节超伸

   膝关节超伸是指腿伸直时膝盖的伸展角度超过 180°，肢骨的顶端相对于股骨向后滑动，使得膝关节向后过度打开，膝关节形成了字母”C”的形状。

骨盆前倾

   骨盆前倾的专业术语叫做“下交叉综合症”，由于长期姿势异常，腹部向前突，臀部后翘，腰部曲度较明显，形成了“前凸后翘”的身体姿态。

解决办法

   其实我们常见脖子疼、腰疼、膝盖疼，大都是由不良体位所引起的。平常不正确的姿势或动作往往会使身体局部出现肌肉不平衡，也就是一部分肌肉(主动肌)长时间处于紧张状态，而另一部分肌肉(拮抗肌)软弱无力，长时间处于拉长状态。
   体态矫正的目的是通过训练，伸展或按摩来修正肌肉失衡所导致的不良体态。其本质就是放松僵硬紧张的肌肉，增强虚弱肌肉的力量，最终达到肌肉的平衡，舒缓或改善因姿势不良引起的疼痛。

驼背、园肩 骨盆前倾 膝超伸
紧张的肌肉 (需要松解，恢复其弹性和伸展性) 胸大肌、胸小肌、肩胛提肌、斜方肌上束、胸锁乳突肌、斜角肌 竖脊肌、腰方肌、髂腰肌、股直肌 小腿三头肌、比目鱼肌、股直肌
虚弱的肌肉 (肌肉处于拉长无力状态，需要加强其力量) 菱形肌、斜方肌中下束、前锯肌、深层颈屈肌 腹直肌、腹内外斜肌、臀部肌群 胫骨前肌、臀部肌群

训练计划

步骤一：改善足底承重

目的 强度 注意 动作
足底筋膜放松 持续 30 秒-1 分钟 足跟与前脚掌之间的位置缓慢滚动，约 5 秒一个滚动单程。

步骤二：拉伸紧张的肌肉

目的 强度 注意 动作
放松胸锁乳突肌 每侧 20 秒 x 3 次 右手将头部缓慢向右侧拉伸，同时左侧肩膀轻轻向下压，保持肩膀固定。
拉伸竖脊肌 持续 20 秒 x 3 次 将腿往上身抱，抱至屁股离开地面，感受下背竖脊肌拉伸。
拉伸髂腰肌 每侧 20 秒 x 3 次 上肢直立，挺胸抬头，重心前移，保持骨盆中立，后侧的腿尽可能向后拉伸。
放松小腿肌肉 每侧 20 秒 x 3 次 跟腱与膝盖之间的位置缓慢滚动，约 5 秒一个滚动单程。
拉伸股四头肌 每侧 20 秒 x 3 次 大腿与地面垂直，膝部顶点指向地面，保持腰部挺直以避免腰椎过伸。

步骤三：增强放松肌肉的力量

目的 强度 注意 动作
激活颈后深层肌肉群 每侧 15 秒 x 3 组 双手拖住头部后侧凸起的位置，下巴微收，头部与手产生对抗。
加强中下斜方肌力量 持续 20 秒 x 3 组 双脚与肩同宽，不要园肩，保持肩膀远离耳朵。
提高下腹肌力量 每组 15 个 x 3 组 保持骨盆后倾，双腿始终悬空。
强化臀部肌群 每组 10 个来回 x 3 组 挺起时，以骨盆带动臀部，双手不要用力，身体和头不要左右偏。
强化胫骨前肌 每侧 10 秒 x 3 次 模拟抬脚尖的过程即可，如果没有弹力带，平地抬脚趾亦可。

步骤四：增强关节稳定性

目的 强度 注意 动作
增强膝关节稳定性 每组 20 次 x 3 组 移动箭步蹲，起立时身体不要前倾，要保持正直，行进中双脚内侧在一条直线上。
增强踝关节稳定性 每侧 20 秒 x 3 次 单脚踮脚站立，右脚脚尖支撑身体，左腿自然向前伸，身体保持不动。

步骤五：纠正动作模式，培养中立位感觉

动作 注意
靠墙浅蹲 腰部贴紧墙壁，致使骨盆后倾，腰凸变平，牵拉了竖脊肌。同时后脑勺和肩胛骨贴近墙面，这样还能矫正园肩和探颈。
倒退上楼梯 保持上半身正直，倒退上楼梯也属于包含伸膝动作但是却不容易产生过伸位置，对于培养关节中立位的意识很有帮助。
