健身计划（三） - 核心和爆发力
发表于 2020-01-30| 更新于 2020-05-19|健身
字数总计:708|阅读时长: 2 分钟
| 阅读量:

   这个健身计划的目标是增强核心力量，增加爆发力，向扣篮梦更进一步。

   为了增加爆发力，这次加入了快速伸缩负荷训练（Plyometric），也叫超等长收缩训练，也叫增强式训练，是增强爆发力的常用手段。
   1月25日，武汉因为疫情封城，北京大大小小的公共场所也都封锁了，大家都只能呆在家里，所以这回没有了健身房，只有家里的哑铃能陪我，所以要使用大重量的力量训练呢，就不能安排了。
   新的健身计划距离上一个健身计划的博客已经有两年多了，这两年并没有按时完成计划，后来身材也走了样，很遗憾。唯一欣慰的是，健身知识在不断的丰富。希望这次家庭健身计划能够坚持执行下去。

周期

   爆发力训练 -> 核心力量训练 -> 休息 （三天一循环）

爆发力训练

热身 Plyometric训练 下肢力量速度训练
脚尖慢跑 ：1分钟 团身跳 ：4组 x 5次 深蹲跳 ：4组 x 3-5次
最伟大的拉伸 ：左右各1次 脚踝跳跃 ：4组 x 5次 哑铃高翻 ：4组 x 3-5次
股四头肌牵拉 ：左右各5步 侧向跳越障碍物 ：4组 x 8次 哑铃抓举 ：4组 x 3-5次
拉膝到胸及踮脚行走 ：左右各5步 跳深 ：4组 x 8次 借力推举 ：3组 x 8-12次
前踢腿行走 ：左右各5步 保加利亚箭步蹲 ：3组 x 5次
侧向蹲 ：左右各5次 负重提踵 ：3组 x 8-12次

核心力量训练

热身 稳定性训练 力量训练
面拉 ：3组 x 8次 土耳其起立 ：左右手 x 2次 单腿硬拉 ：3组 x 8次
交替单手平板支撑 ：两侧各15个 对角俯卧支撑 ：左右手 x 30秒 仰卧对角交替收膝 ：2组 x 20次
动态侧平板支撑 ：两侧各30秒 农夫走 ：1分钟 侧卧腹斜肌卷腹 ：2组 x 左右15次
俯卧直腿后摆 ：两侧各10次 哑铃砍劈 ：左右手 x 15次 仰卧直腿举腿 ：2组 x 20次
单腿仰卧挺髋 ：左右手 x 8次 哑铃三威胁 ：左右手 x 15次 侧卧夹球卷腹 ：2组 x 左右15次

参考

[2]王卫星,李海肖.竞技运动员的核心力量训练研究[J].北京体育大学学报,2007(08):1119-1121+1131.
