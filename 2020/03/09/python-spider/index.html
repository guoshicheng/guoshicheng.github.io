python简单爬虫练习 - 梨泰院class漫画下载
发表于 2020-03-09| 更新于 2020-05-19|编程
字数总计:897|阅读时长: 3 分钟
|阅读量:

   最近漫改的韩剧《梨泰院Class》还蛮火的，追了一下，最近更新的第11、12集剧情太扯了，支线黑人服务员和房东老太太根本就是剧情需要。所以想去看看原著漫画的剧情，古风漫画网可以免费观看，但是广告太多了，图片排版也很别扭，最后打算写个爬虫把图片下载到本地看。

   首先从整体看，我们需要这几个步骤：
     -> 1. 获取网页html源代码
     -> 2. 分析图片代码和图片地址
     -> 3. 下载图片到本地
     -> 4. 获取下一章节url
     -> 5. 回到第一步

   接下来分别实现，第一步获取网页html源代码，使用python的requests库，因为网站会检测访问行为是爬虫还是浏览器，所以设置下请求头域——Header，模仿浏览器访问。

   第二步分析图片代码和图片地址,图片地址由三部分组成:
     头部：https://res.xiaoqinre.com
     中段：/images/comic/589/1176126
     结尾：/157804675608omNrJSJJ8gmZZ3.jpg
   头部是固定的，中段和结尾都可以在源代码body的第一个script标签里找到，使用BeautifulSoup找到标签，再使用正则表达式匹配就可以了。（推荐一个正则表达式测试网站Regex101）

   第三步下载图片到本地，章节名称也可以通过正则筛选出来，最后使用os库检查本地路径是否存在，二进制写入文件就可以保存图片了。

   第四步获取下一章节链接，同第二步，用正则表达式匹配，也可以用beautifulSoup获取下一章按钮的链接。

   第五步判断是否有下一章，有就循环下载，没有就退出。
   完整代码在这里 ： github-downloadItaewonClass.py
   最后想说一下，这次下载的漫画是盗版的，梨泰院Class应该是腾讯买了版权做的翻译。在腾讯漫画网看是需要付费的，一个章节68点券，一共65章节，大概45Q币，经济上有能力又不想看盗版的，可以支持一下正版。还有就是大家下载漫画之后，自己看看无所谓，不要放到网上进行传播。
