篮球基本功 - 脚步训练

发表于2020-10-12|更新于2020-11-02|篮球
|字数总计:734|阅读时长:2分钟

训练计划

热身

动作 注释
球绕身 移动过半场，顺/逆时针交换
指尖运球 绕腿8字，反8字，共10圈
胯下 + 体前 + 背后 半场运球一来回
胯下体前 + 双胯下 + 胯下背后 左右 * 10次

第一组：坠步(Drop Step) + 口袋运球(Pocket Dribble)

   口袋运球就是将球拉回口袋位置的运球，它可以为接下来的传球，突破，偷懒提供更快速的动作，目前是最常见的运球方式。口袋运球时，手指向下或者向外，不要向上这样不好运球，向外是要注意不要翻腕停球，需要多加练习。

动作 注释
左右运球 + 口袋运球（坠步） 身体对侧拍球拉回做口袋运球
口袋运球（坠步） + 摸水瓶 左右手各20次
口袋运球（坠步） + 胯下 同上
口袋运球（坠步） + Break-in + 背后拉回 同上
口袋运球（坠步） + 顺步push + 胯下拉回 同上

第二组：拖曳步 (Shuffle Step)

动作 注释
拖曳步 + 体前
拖曳步 + Killer Crossover 如何防守跟上，但没有贴紧，胯下体前突破，左右手各20次
拖曳步 + 背后运球 如何防守跟上，并且尝试断球，背后横向变向再突破

第三组：行进中基础运球

动作 注释
行进中 + Treadmill 先右手45°运球，然后反方向90°变向
行进中 + 变速攻击步 在做Treadmill时降速停顿，然后后退立即蹬地加速。
行进中 + Break-in
行进中 + 背后变向
行进中 + 背后环绕运球
行进中 + 360°转身

第四组：行进中刺探步(Jab Step)

   欺骗防守人，佯装突破。实则蹬地，加速反方向突破。

动作 注释
行进中 + 体前变向（同时反向刺探步）
行进中 + 刺探步 + 双体前
行进中 + 刺探步 + Killer Crossover

第五组：犹豫步(Fake Hesitation)

通过瞬间的停球，假装收球准备投篮，诱使防守球员重心上升，随后立即加速突破。
重点：急停、眼睛瞄篮、重心降低.

动作 注释
Fake Hesitation 重心从前脚到后脚，在蹬地突破
hang dribble 运球过程中停顿，突然蹬地突破
pull back + Hand Fake + In & Out 急刹减速，假装收球投篮

第六组：组合动作

动作 注释
点地胯下 + 刺探步 + 体前变向 点地胯下高重心，刺探步快速降低重心同事体前变向
双跨下（接Foot Switch） + 坠步顺步突破
双跨下（接Foot Switch） + 坠步2-1干拔
In & Out（*2）+ 体前/360°转身
In & Out/胯下/背后变向 + 背后环绕运球（+ 体前变向）

三步上篮

动作 注释
头顶绕球 + overhand/underhand lay-up
头顶绕球（+跳步） + overhand/underhand lay-up
头顶绕球（+欧洲步） + overhand/underhand lay-up
腰间绕球 + overhand/underhand lay-up
腰间绕球（+跳步） + overhand/underhand lay-up
