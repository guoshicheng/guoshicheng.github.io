现阶段篮球训练计划

发表于2021-04-23|更新于2021-06-12|篮球
|字数总计:810|阅读时长:2分钟

训练计划

Part1.准备活动

动作
热身 正反跑 => 高抬腿交叉步 => 防守滑步 => 防守交叉步 => 分腿突破
动态拉伸 踢腿 => 拉伸大腿后侧 => 拉膝到胸 => 股四头肌牵拉 => 髋关节灵活度 => 最伟大拉伸
关节灵活度 髋关节 => 踝关节 => 肩关节
协调性 单双脚快速弹跳: 横行 => 纵向

Part2.运球训练

动作
热身 弓箭步球绕身 => 指尖绕腿运球
原地运球 垂直运球 => In & Out => 左右运球 => 背后运球
过半场运球 反胯下 => 背后 => 横向 In & Out => 后退 In & Out
Z 字运球 垂直运球衔接: 胯下 => 背后
坠步衔接: Crossover => 推球变向
In & Out 衔接: 体前 => 胯下 => 背后
变向误导 体前刺探步 => In & Out 刺探步
重心变化 胯下体前 => 胯下背后 => 双胯下 => 双体前 => 双背后
节奏运球 体前拖曳步 + 双体前 => 胯下拖曳步 + 双体前 => 背后拖曳步 + 双体前
背后环绕式运球 垂直运球 + 大背运 => In & Out + 大背运
胯下 + 大背运 + 体前 => 背后 + 大背运 + 体前
急停 同手同脚(+ 重置胯下) => 体前/胯下/背后

Part3.上篮训练

动作
麦肯训练 内外侧脚和内外侧手的 4 种组合 X 面(背)框
终结能力 三步上篮 => 三步反篮
脚步练习 同手同脚 => 欧洲步 => 双脚并步 => Rondo 上篮 => 转身上篮

Part4.突破训练

动作
三威胁 + 一次运球变向 + 垫步投篮 AA 步 + 背后变向
Crossover 交叉步 + 胯下变向
转身 + 体前变向
两次运球 + 一次运球变向 + 欧洲步终结 胯下背运 + 背后变向
体前胯下 + 胯下变向
坠步推球 + 体前变向
突破急停 + 转髋 + 后撤步投篮 胯下拉回 + 垂直运球 + 背后变向
背后拉回 + 垂直运球 + 胯下变向

Part5.投篮训练

动作
投篮手型 从 3 秒开始,原地投中一个空心,就像后退一步,不尽则再次尝试
合球起球 胯下/背后衔接: 垫步合球 => 拖曳步 + 一二步合球
接球投篮 侧身接球一二步 => 90° 垫步转身
一二步跳投 垂直运球/胯下/背后衔接: 交叉步突破 + 一二步合球
垂直运球/胯下/背后衔接: 拖曳步突破 + 一二步合球
垂直运球衔接: 后撤步拉回 + 一二步合球
零一二跳投 变向/同手同脚衔接:后撤步拉回 + 一二步合球
炸球衔接:后撤步拉回 + 一二步合球(利拉德)
In & Out 衔接:交叉步拉回 + 垫步合球
垫步跳投 垂直运球/胯下/背后衔接: 后撤步拉回 + 垫步合球
急停衔接: 各种拉回 + 垫步合球
抛投 单脚: 一次运球 => 左右侧接球
双脚: 一二步 => 垫步 => 转身抛投 => 跳步
